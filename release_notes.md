# WinMonitor 1.0

WinMonitor has reached version 1.0 and is now officially open-source!

## Features
- **Real-Time Data:** View CPU%, RAM%, GPU%, network speeds (Upload/Download), and per-drive disk performance active times.
- **Customizable Visibility:** Toggle individual modules (CPU, RAM, Disk, GPU, Network) on/off directly from the right-click context menu.
- **Always on Top & Draggable:** Drag to position anywhere. Auto-resizes from the left side, keeping the right edge fixed.
- **Temperature Monitoring:** Displays CPU package and GPU core temperatures.
- **Auto-Run:** "Start with Windows" toggle available in the context menu.
- **Configuration Persistence:** Settings are saved locally and persist across runs.
- **Lightweight:** Minimal CPU and RAM footprint.

## What's New
- **Open Source:** Released under the MIT License.
- **Removed Tray Icon:** To reduce redundancy and eliminate taskbar clutter, the system tray icon has been removed. All configuration and exit options are now available directly by right-clicking the floating info bar itself.
- **Custom Project Icon:** The executable now sports a custom monitor-style icon.
- **GitHub Actions:** Added automated build pipelines to automatically compile and release Windows executables for future updates.

## Installation
Download `WinMonitor.exe` attached to this release and run it directly. No installation is required.
