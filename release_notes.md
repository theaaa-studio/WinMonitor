# WinMonitor 1.0

WinMonitor has reached version 1.0 and is now officially open-source!

## What's New
- **Open Source:** Released under the MIT License.
- **Removed Tray Icon:** To reduce redundancy and eliminate taskbar clutter, the system tray icon has been removed. All configuration and exit options are now available directly by right-clicking the floating info bar itself.
- **Custom Project Icon:** The executable now sports a custom monitor-style icon.
- **GitHub Actions:** Added automated build pipelines to automatically compile and release Windows executables for future updates.

## Installation
Download the WinMonitor.exe attached to this release and run it. No installation required.
